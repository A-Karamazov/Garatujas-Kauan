# Garatujas do Kauan

HTML (HyperText Markup Language) é a linguagem de marcação usada para estruturar o conteúdo dos sites e
páginas da web usando tags com funções diferentes. Um arquivo HTML é dividido em head (meta informações)
e body (estrutura visível da página)

CSS (Cascading Style Sheets) é a linguagem que estiliza uma página HTML. Por ela, se controla a estética
da página web, controlando cores, tamanhos, posições e outras propriedades dos elementos da página 

JavaScript (JS) é a linguagem que adiciona interatividade às páginas. Ele permite criar comportamentos
dinâmicos, como responder a cliques, validar formulários, alterar conteúdos e criar animações.

Java é uma linguagem de programação robusta e estruturada, muito usada no desenvolvimento de sistemas maiores, 
aplicativos e softwares corporativos. Ela possui tipagem forte e precisa ser compilada antes de ser executada.

JavaScript é uma linguagem voltada principalmente para a web, executada diretamente no navegador. 
Diferente do Java, é mais flexível e amplamente utilizada para tornar sites interativos.

TypeScript (TS) é uma extensão do JavaScript que adiciona tipagem e maior controle sobre o código. Ele ajuda
a evitar erros e facilita a organização em projetos maiores, sendo convertido para JavaScript ao final.

POO (Programação Orientada a Objetos) é um paradigma de programação que organiza o código com base em objetos, representando 
elementos do mundo real dentro do sistema. Isso torna o desenvolvimento mais estruturado, reutilizável e fácil de manter.

Classe é uma estrutura que funciona como um molde para criar objetos. 
Ela define quais características e comportamentos os objetos terão.

Objeto é uma instância criada a partir de uma classe. Ele representa algo concreto 
dentro do programa, possuindo seus próprios valores e podendo executar ações.

Atributo é uma característica de um objeto. Ele armazena informações 
que descrevem seu estado, como nome, idade ou cor.

Método é uma ação que um objeto pode executar. Ele define comportamentos 
e funcionalidades que podem ser usados dentro da classe.

Getters e Setters são métodos especiais utilizados para acessar e modificar atributos de forma 
controlada. Eles ajudam a proteger os dados e permitem validações antes de alterações.

Construtor é um método especial executado automaticamente quando um objeto é criado. 
Sua função é inicializar os atributos com valores iniciais.

Herança é o mecanismo que permite que uma classe herde características e métodos de outra. 
Isso facilita o reaproveitamento de código e a organização do sistema.

Encapsulamento é o princípio de proteger os dados internos de um objeto, permitindo 
acesso apenas por meios controlados. Isso aumenta a segurança e evita alterações indevidas.

Polimorfismo é a capacidade de um mesmo método ter diferentes comportamentos dependendo do 
objeto que o utiliza. Esse conceito torna o código mais flexível e adaptável.
